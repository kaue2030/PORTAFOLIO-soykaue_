# Estudio Bíblico - Cristología

Plataforma de estudio bíblico profundo centrada en la Cristología, explorando la persona y obra de Jesucristo. Un espacio para crecer en la fe, entender la Biblia y conocer más a Cristo.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `npm run dev`
